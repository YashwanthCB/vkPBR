# vkPBR

Developed a vulkan based PBR renderer using the Cook–Torrance BRDF with GGX distribution, Smith geometry term, and Schlick Fresnel under inverse-square attenuated point lights in C++.

Demo: [Youtube](https://www.youtube.com/watch?v=eXA3O2OZgcw)
